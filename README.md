## Installation

To clone and run this project locally, use the following commands:

```bash
# Clone this repository
git clone https://github.com/barishizm/barishizm.github.io.git

# Navigate to the project directory
cd barishizm.github.io

# Install local build tooling
npm install

# Generate optimized static output
npm run build

# Verify generated output
npm run verify
```

## Build Workflow

- Authoring HTML lives in `src/`
- Generated deploy assets are written to `assets/build/`
- Optimized deploy pages are written back to the repo root for GitHub Pages

Useful commands:

```bash
npm run build
npm run build:images
npm run build:css
npm run build:js
npm run build:html
npm run verify
```
